Berikut ini adalah open data yang divisualisasikan dalam peta menggunakan OpenLayers :

  1. Data garis batas pantai Pulau Damar diambil dari OpenStreetmap menggunakan Overpass API. Lisensi dari data ini adalah Open Data Commons Open Database License (ODbL).
  2. Data posisi gunung di Pulau Damar diambil dari OpenStreetmap menggunakan Overpass API. Lisensi dari data ini adalah Open Data Commons Open Database License (ODbL).
  3. Data lahan kritis di Pulau Damar diambil dari Kementrian Kehutanan. Lisensi dari data ini adalah Creative Commons Attribution Lisence (cc-by) yang mana disebutkan di data.go.id.
